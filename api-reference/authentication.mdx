---
title: "Authentication"
description: "Comment authentifier les requêtes vers l'API ?"
---

Nous allons voir ici comment authentifier les requêtes vers l'API.

Pour cela, il faut utiliser un token d'authentification. Lors de vos requêtes, vous devez ajouter un header `Authorization` avec la valeur `Bearer <token>`.

```bash
'Authorization': 'Bearer <team_token>'
```

## Token API

Vous pouvez trouver votre token API dans la page API de votre espace membre.
