---
title: "Lister les annonces"
api: "GET https://api.houzeo.fr/real-estate/search"
description: "Cette route permet de lister les annonces"
---

### Body

<ParamField body="page" type="integer" default="1" required={false} optional={true} placeholder={1}>
  Numéro de la page à afficher
</ParamField>

<ParamField body="limit" type="integer" default="50" required={false} optional={true}>
  Nombre d'annonces par page
</ParamField>

<ParamField body="sortBy" type="string" required={false} optional={true} default={"created_at"}>
  Le champ sur lequel trier les résultats. Les valeurs possibles sont : price, price_per_sqm, surface, rooms, created_at, updated_at
</ParamField>

<ParamField body="order" type="string" required={false} optional={true} default={"desc"}>
  L'ordre de tri. Les valeurs possibles sont : asc, desc
</ParamField>

<ParamField body="types" type="array" required={false} optional={true}>
  Le type d'annonces à retourner : sale, rent
</ParamField>

<ParamField body="origins" type="array" required={false}>
  Les plateformes sur lesquelles les annonces sont publiées : A FAIRE
</ParamField>

<ParamField body="categories" type="array" required={false}>
  Les plateformes sur lesquelles les annonces sont publiées : apartment, house, office, land, commercial, parking, other
</ParamField>

<ParamField body="owner_types" type="array" required={false}>
  Les plateformes sur lesquelles les annonces sont publiées : individual, professional
</ParamField>

<ParamField body="keywords" type="array" required={false}>
  Liste des mots clés à rechercher dans les annonces
</ParamField>

<ParamField body="keywords_operator" type="string" required={false}>
  Les opérateurs de recherche de mots clés, si plusieurs mots clés sont fournis : and, or
</ParamField>

<ParamField body="price_min" type="number" required={false}>
  Prix minimum
</ParamField>

<ParamField body="price_max" type="number" required={false}>
  Prix maximum
</ParamField>

<ParamField body="price_by_sqm_min" type="number" required={false}>
  Prix minimum au m²
</ParamField>

<ParamField body="price_by_sqm_max" type="number" required={false}>
  Prix maximum au m²
</ParamField>

<ParamField body="surface_min" type="number" required={false}>
  Surface minimum
</ParamField>

<ParamField body="surface_max" type="number" required={false}>
  Surface maximum
</ParamField>

<ParamField body="rooms_min" type="number" required={false}>
  Nombre de pièces minimum
</ParamField>

<ParamField body="rooms_max" type="number" required={false}>
  Nombre de pièces maximum
</ParamField>

<ParamField body="bedrooms_min" type="number" required={false}>
  Nombre de chambres minimum
</ParamField>

<ParamField body="bedrooms_max" type="number" required={false}>
  Nombre de chambres maximum
</ParamField>

<ParamField body="floor_min" type="number" required={false}>
  Nombre d'étages minimum
</ParamField>

<ParamField body="floor_max" type="number" required={false}>
  Nombre d'étages maximum
</ParamField>

<ParamField body="building_floor_min" type="number" required={false}>
  Nombre d'étages de l'immeuble minimum
</ParamField>

<ParamField body="building_floor_max" type="number" required={false}>
  Nombre d'étages de l'immeuble maximum
</ParamField>

<ParamField body="construction_year_min" type="number" required={false}>
  Année de construction minimum
</ParamField>

<ParamField body="construction_year_max" type="number" required={false}>
  Année de construction maximum
</ParamField>

<ParamField body="dpe_value_min" type="number" required={false}>
  Valeur du diagnostic de performance énergétique minimum
</ParamField>

<ParamField body="dpe_value_max" type="number" required={false}>
  Valeur du diagnostic de performance énergétique maximum
</ParamField>

<ParamField body="ges_value_min" type="number" required={false}>
  Valeur du diagnotic de gaz à effet de serre minimum
</ParamField>

<ParamField body="ges_value_max" type="number" required={false}>
  Valeur du diagnotic de gaz à effet de serre maximum
</ParamField>

<ParamField body="keywords" type="array" required={false}>
  Options et caractéristiques du bien : fibre_optique, meuble, non_meuble, ascenceur, sans_ascenceur, jardin, terrasse, garage, parking, cave, piscine, interphone, terrasse, digicode, rdc
</ParamField>

### Response

<ResponseField name="totalDocs" type="number">
  Nombre total d'annonces correspondant à la requête
</ResponseField>

<ResponseField name="limit" type="number">
  Limite de résultats par page
</ResponseField>

<ResponseField name="totalPages" type="number">
  Nombre total de pages
</ResponseField>

<ResponseField name="page" type="number">
  Page courante
</ResponseField>

  This is the internal ID for this user group. You don't need to record this information, since
  you will not need to use it.
</ResponseField>

<ResponseField name="token" type="string">
  This is the user group token (userGroupToken or USER_GROUP_TOKEN) that will be used to identify
  which user group is viewing the dashboard. You should save this on your end to use when rendering
  an embedded dashboard.
</ResponseField>

<ResponseField name="name" type="string">
  This is the name of the user group provided in the request body.
</ResponseField>

<ResponseField name="provided_id" type="string">
  This is the user_group_id provided in the request body.
</ResponseField>

<ResponseField name="properties" type="JSON or null">
  This is the properties object if it was provided in the request body
</ResponseField>

</Expandable>
